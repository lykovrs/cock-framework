demo-button works! 

  Используется через добавления атрибута "sc-button" к элементу:
  <button sc-button>
    Кнопка
  </button>

  По умолчанию добавляется класс `.sc-button`.
  Может иметь атрибут модификатор "mod"="value", которые добавит соответствующий класс `.sc-button_[mod.value]`.
  Может иметь атрибут "size"="value", которые добавит соответствующий класс `.sc-button_[size.value]`.

  Может содержать опциональные svg иконки с атрибутом указания места иконки [after, before] относительно текста:
  <button sc-button>
    Кнопка
    <sc-svg-icon after src="sc/svg/delete.svg"></sc-svg-icon>
    <sc-svg-icon before src="sc/svg/delete.svg"></sc-svg-icon>
  </button>
